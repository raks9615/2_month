Команды для регистрации в git:
git config --global user.name "Your name"
git config --global user.email "your-email@example.com"

Команды только для одного проекта:
git init

Команда вставляет нашу ссылку в наш репозиторий:
git remote add origin https://github.com/raks9615/2_month.git

Основные команды в Git

1) Проверка состояния файлов в проекте
git status

2) Команда для добавления файлов в буфер новых файлов и изменений в файлах.
После команнды add через пробел ставите точку чтобы добавить все файлы. Все выделяет и копирует
git add .

Сохраняем все файлы
git commit -m "название сохранения"

git push origin master
